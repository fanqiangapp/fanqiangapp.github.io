三、手把手安装配置教程

3.1 准备工作

首先确保你的Chrome浏览器是最新版本（目前最新为v115+）。点击浏览器右上角的三个点>帮助>关于Google Chrome即可检查更新。我建议在安装前先清理浏览器缓存，这能避免一些潜在的兼容性问题。

3.2 详细安装步骤

  1. 打开Chrome网上应用店（可直接搜索"Chrome Web Store"）
  2. 在搜索框输入"蔷薇科学上网"或"Rose VPN"
  3. 找到官方扩展（注意识别开发者信息，避免安装山寨版本）
  4. 点击"添加到Chrome"按钮
  5. 在弹出的权限请求对话框中点击"添加扩展程序"

专业提示：如果无法访问Chrome应用商店，可以尝试从开发者官网下载CRX文件，然后通过"开发者模式"手动安装。

3.3 首次配置指南

安装完成后，浏览器右上角会出现蔷薇图标。点击后你会看到一个简洁的控制面板：

  • 服务器选择：建议优先选择地理位置较近的节点（如日本、新加坡对亚洲用户较友好）
  • 协议设置：高级用户可切换WireGuard或OpenVPN协议
  • 自动连接：可设置特定网站自动启用代理

我个人的配置经验是：白天使用日本节点获得低延迟，晚上切换至美国节点获取更稳定带宽。对于视频流媒体，建议选择专用流媒体优化节点。

四、高级使用技巧与性能优化

4.1 分流策略配置

通过右键点击扩展图标>选项，可以设置分流规则： - 全局代理：所有流量通过VPN - 智能分流：仅国外网站走代理 - 自定义规则：指定域名列表

我通常选择智能分流模式，这样国内网站依然保持直连速度，而国际流量得到保护。

4.2 多账号管理

支持同时添加多个订阅账号，通过点击用户头像切换。这对需要区分工作和个人使用的用户特别有用。

4.3 速度优化技巧

  • 在拥挤时段尝试较少人使用的服务器
  • 更换连接协议（WireGuard通常最快）
  • 禁用其他占用带宽的扩展
  • 清理浏览器缓存和cookies

FAQ

Mihomo 如何启用 TUN 模式？
在 Mihomo 的配置文件中启用 `tun:` 模块并安装系统驱动即可实现全局代理。TUN 模式可将所有系统流量导向代理核心，实现真正意义上的全局加速。
Nekobox 如何管理多协议节点和分流？
Nekobox 可同时管理 Shadowsocks、VMess、Trojan 等多协议节点。结合策略组和分流规则，可实现不同流量走不同节点，提升网络效率和管理便利性。
SSR 的协议参数和混淆参数如何配置最安全？
选择 AEAD 或协议插件可提升安全性，混淆方式如 tls1.2_ticket_auth 可以模拟正常 HTTPS 流量。合理搭配协议和混淆参数，能够有效抵御 DPI 检测和流量封锁。
VLESS + XTLS 如何降低移动网络延迟？
XTLS 通过减少握手次数和加密开销降低延迟。结合策略组和多节点切换，可在高延迟或不稳定的移动网络环境中保持连接稳定，提升网页浏览、视频和游戏体验流畅度。
QuantumultX 如何实现按端口和协议分流？
QuantumultX 支持按 TCP、UDP、端口及应用协议分流流量，结合策略组选择最优节点。这样可以为游戏、视频和广告屏蔽提供最佳网络体验，提高访问速度和稳定性。
Sing-box 如何实现域名精确分流？
Sing-box 支持按域名匹配流量，将不同应用流量分配到指定节点。结合策略组使用，可自动选择最优节点，提高访问速度和网络稳定性，实现精确流量管理。
Surfboard 的规则文件如何支持广告屏蔽？
规则文件可添加广告域名或 URL 列表，将请求重定向或阻断。结合策略组和自动更新，Surfboard 可实现全局广告屏蔽，提高浏览体验并节省带宽。
Sing-box 支持按 IP 段分流吗？
支持。Sing-box 可根据 IP 段将国内流量直连，国外流量走代理，结合策略组实现智能分流。这样可保证访问速度和稳定性，同时优化节点使用效率，提高网络体验。
V2RayN 的 GUI 界面如何导出节点？
V2RayN 提供节点导出功能，可生成分享链接或二维码。用户右键节点选择“分享”即可导出，以便在其他设备导入同样的代理配置，保持同步使用体验。
Mihomo 节点分组管理有哪些优势？
节点可按地区、延迟或用途分类，结合策略组实现自动分流。便于管理大量节点，快速选择最优节点，提高访问速度和连接稳定性，同时优化维护效率和网络体验。
